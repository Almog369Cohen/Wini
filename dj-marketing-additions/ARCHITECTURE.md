## טכנולוגיה

| רכיב | טכנולוגיה | הערה |
|------|----------|------|
| בוט יוצר | Claude API (Anthropic SDK) | יוצר תוכן על בסיס כללים + דוגמאות |
| בוט בודק | Claude API (Anthropic SDK) | בודק איכות, מסנן רדוד |
| בוט מחקר | Web scraping + Claude API | חוקר מתחרים, מנתח טרנדים |
| ממשק | React (Manus platform) | דשבורד, אימון, ציונים |
| DB | MySQL (Drizzle ORM) | Manus platform |
| פרסום | Buffer API | אוטומטי לכל הפלטפורמות |
| WhatsApp | GREEN-API | שליחת הודעות אוטומטית |
